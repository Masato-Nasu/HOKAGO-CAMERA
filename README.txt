放課後CAMERA v10

最小構成で作り直した版です。

同梱:
- index.html
- sw.js
- manifest.webmanifest
- icon-192.png
- icon-512.png
- apple-touch-icon.png
- favicon-32.png

使い方:
1. スマホでは index.html を HTTPS で公開して開いてください。
2. 「カメラで撮る」または「写真を選ぶ」で画像を入れます。
3. 「文字」または「スタンプ」タブで追加します。
4. 写真上の文字/スタンプをタップすると「調整」タブで編集できます。
5. 「画像で保存」でPNG保存できます。
6. 「データを保存」で、このアプリで再生するための .hokago データを保存します。
7. 「データを開く」で受け取った .hokago データを読み込めます。

注意:
- PWA/カメラ/共有は HTTPS 配信時に安定します。
- 古いPWAが残っていると古い画面を掴むことがあります。古いホーム画面アイコンを消してから再追加してください。
